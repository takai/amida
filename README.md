# amida.rb

amida.rb is an amida generator.

## usage

```
ruby -Ilib bin/app.rb [options]

options:

    -c, --column      Number of columns
    -r, --row         Number of rows
    -h, --help        Print this message
```

```
|  |  |  |  |
|--|  |  |  |
|  |--|  |  |

```
